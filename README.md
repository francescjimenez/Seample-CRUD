## Available Scripts

In the project directory, you can run:
### `npm start`
Runs the app in production mode mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### For development
You need to open two consoles.<br>

#### Client
npm run start:client

#### Server
npm run dev:server
